\hh{Использование удалённой компиляции}\n\h{ecfremc}\n\tОсуществляет сборку проекта Engine Runtime на другом узле. Синтаксис:
\n\code{ecfremc <PROJECT> <NODE> [-a <ARCH>] [-o <OS>] [-r | -d] [-i <IP>] [-p <PORT>]}
\n PROJECT - файл проекта для сборки.
\n NODE - узел, на котором будет осуществляться компиляция.
Может быть указано: имя узла, полный адрес сервера (16 шестнадцатеричных цифр) или индекс узла (4 шестнадцатеричные цифры).
\n i - переопределяет IP адрес узла, через который будет направлен запрос. Значение по умолчанию: 127.0.0.1.
\n p - переопределяет порт сервера узла. Значение по умолчанию 10666.
\n a - указывает архитектуру машины для компиляции.
\n o - указывает целевую ОС для компиляции (если целевой узел поддерживает сборку под разные ОС).
\n r - использует релизную конфигурацию.
\n d - использует конфигурацию отладки.
