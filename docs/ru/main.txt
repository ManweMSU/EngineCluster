\hh{Engine Cluster Framework}\n\tEngine Cluster Framework - комплекс программного обеспечения,
предназначенный для объединения ПК в пределах локальной сети в вычислительный кластер.
Комплекс ПО включает в себя следующие продукты:\n
- Сервер кластера - программа, реализующая сервер узла сети, графический интерфейс настройки, предоставляет документацию.\n
- Инструменты кластера - набор программ командной строки для управления отдельными службами кластера (устанавливается отдельно).\n
- Пакеты исходного кода - модули для Engine Runtime, позволяющие разрабатывать ПО, способное взаимодействовать в рамках кластера (устанавливается отдельно).\n
- Fair Blur - программа, демонстрирующая распределённые вычисления на примере эффекта гауссова размытия (устанавливается отдельно).
