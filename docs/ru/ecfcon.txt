\hh{Работа с распределённой консолью}\n\h{ecfcon}\n\tВыводит сообщения, отправляемые через распределённую консоль. Синтаксис:
\n\code{ecfcon [-i <IP>] [-p <PORT>] [-s] [-t] [-o <FILENAME>]}
\n i - переопределяет IP адрес узла для получения сообщений. Значение по умолчанию: 127.0.0.1.
\n p - переопределяет порт сервера узла. Значение по умолчанию 10666.
\n s - предписывает не печатать информацию об отправителе.
\n t - предписывает печатать время возникновения сообщения.
\n o - дублирует вывод в указанный текстовый файл. Можно указать произвольное число файлов.
\n\h{ecfconwr}\n\tИнтерактивная программа, позволяющая отправлять сообщения в распределённую консоль.
Для выхода следует ввести пустую строку. Синтаксис:
\n\code{ecfconwr [-i <IP>] [-p <PORT>]}
\n i - переопределяет IP адрес узла для отправки сообщений. Значение по умолчанию: 127.0.0.1.
\n p - переопределяет порт сервера узла. Значение по умолчанию 10666.
