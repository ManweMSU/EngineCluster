\hh{Ручной запуск распределённых вычислений}\n\h{ecfexec}\n\tОсуществляет ручной запуск распределённых вычислений. Синтаксис:
\n\code{ecfexec <PACKAGE> [-a <NAME> <VALUE>] [-f <INPUT>] [-o <OUTPUT>] [-i <IP>] [-p <PORT>]}
\n PACKAGE - файл пакета с ядром вычислений.
\n i - переопределяет IP адрес узла, через который будет направлен запрос. Значение по умолчанию: 127.0.0.1.
\n p - переопределяет порт сервера узла. Значение по умолчанию 10666.
\n f - использует произвольный файл в качестве входных данных вычисления.
\n a - использует в качестве входных данных структуру с рефлексией.
В структуре будет поле с заданным именем и значением типа Engine::ImmutableString. Может быть указано несколько таких полей.
\n o - указывает файл, в который будет сохранён двоичный вывод вычислений.
