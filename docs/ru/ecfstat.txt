\hh{Получение информации о сети}\n\h{ecfstat}\n\tВыводит информацию о кластерной сети. Синтаксис:
\n\code{ecfstat [-i <IP>] [-p <PORT>] [-o] [-e] [-s]}
\n i - переопределяет IP адрес узла, через который будет запрошена информация. Значение по умолчанию: 127.0.0.1.
\n p - переопределяет порт сервера узла. Значение по умолчанию 10666.
\n o - печатает в том числе информацию о неактивных в данных момент узлах.
\n e - печатает для каждого активного узла информацию о его клиентах.
\n s - печатает для каждого активного узла информацию о питании и прогрессе выполнения распределённой задачи.
