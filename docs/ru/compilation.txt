\hh{Удалённая компиляция}\n\tУдалённая компиляция - служба, реализуемая серверами для сборки ПО средствами другого узла кластера.
Для работы службы необходим полностью настроенный Engine Runtime версии 1.3 и выше.
Со стороны клиента обращение к службе осуществляется через программу командной строки ecfremc.
Текстовый вывод удалённо запущенного сборочного конвейера осуществляется через распределённую консоль.
